当院では、歯医者とは痛くなる前に気軽に通うところであるという考えのもと、予防歯科に力を入れて取り組んでいます。

むし歯や歯周病の治療と再発を繰り返さないよう、当院では治療よりも予防を重視しています。わからないことや気になることなど、なんでもお気軽にご相談ください。

大阪府四條畷市の歯医者「河野歯科」のホームページへお越しいただきありがとうございます。小さなお子さんからご高齢の方まで安心して通える地域密着型の医院として親身で丁寧な対応を心がけています。また、JR片町線忍ヶ丘駅から徒歩3分の立地で、駐車場完備のため通院に便利なうえ、平日は20時まで診療しています。お仕事や学校帰りの通院も可能です。

当院は診療においては「予防歯科」を重視しています。むし歯や歯周病になった際にきちんと治療することも大切ですが、もっと大切なのはむし歯や歯周病にならないよう定期的に予防を行うことです。歯の治療は、身体的にも心理的にも負担がかかるものです。特にお子さんに負担をかけないようにするためにも、ぜひ、親子で一緒に予防に取り組みましょう。当院には女性ドクターも在籍しておりますので、マタニティ歯科でのご相談など、ご希望の場合は遠慮なくお問い合わせください。
